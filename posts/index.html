  • Fedora Keyboard

    When switching from a mac to linux installed in a windows machine I suggest to swap the left control key with the left alt key. On Fedora 32 it can be done by accessing the Tweaks > Keyboard & Mouse > Additional Layout Options.

  • Github SSH Keys

    Adding a public ssh key to Github allow us to perform git commands without having to type in our password to authenticate. For this to work we need to make sure that our repo is set to use ssh and that Github has our public key.
